Conversor de Moedas

Este projeto foi desenvolvido como parte de um trabalho acadêmico para estudar a utilização do Angular e integrar uma API de conversão de moedas.

A aplicação utiliza a Exchangerate API para obter as taxas de câmbio em tempo real entre diversas moedas.

Funcionalidades principais:
  • Conversão direta entre duas moedas selecionadas
  • Exibição de taxas de câmbio atualizadas
  • Integração com a API de moedas para obter dados em tempo real
Objetivo: Aprender os conceitos fundamentais do Angular, como componentes, serviços e rotas, e aplicar boas práticas de integração com APIs externas.
